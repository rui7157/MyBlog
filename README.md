# 个人博客站


> ### Flask 制作的个人博客站

* 简介
    博客站非常简洁使用“Black&White”前端模版，非常适合个人博客站，前端响应式设适合手机和PC端浏览,后台有管理页面编辑、删除、发布
后端的真实登录地址为:127.0.0.1:5000/adm/lr ，网站采用MySQL建立数据库。搭建时候需要sqlalchemy 命令“create_all”创建数据库。
